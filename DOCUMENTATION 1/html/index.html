How to deploy a website so everyone can see it

01

Firstly create a Netlify account

Netlify is a free web app that allows users to deploy websites with a domain name and https safety lock. Many frontend developers use Netlify because it's easy to use, practical and also it's linked to Git account

02

Add your site to Netlify

Import an existing project form a Git repository, pick a starter template or just deploy manually by dragging your folder of HTML, CSS and JS files to box at the bottom of the page

03

Create a domain name

You can choose your own domain name by just editing page URL. Once you do that, it will be easier for everyone to access your page by just typing it in google searchbar.
